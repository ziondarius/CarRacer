# Car Racer

A lightweight browser-based car racing game built with HTML, CSS, and vanilla JavaScript.

## How to run

1. Open `index.html` in a browser.
2. Click **Start Game**.

## Controls

- Keyboard: `Left / Right Arrow` or `A / D`
- Mobile: on-screen `Left` and `Right` buttons

## Gameplay

- Avoid enemy traffic.
- Score increases over time.
- Speed ramps up as your score climbs.
- Best score is saved in browser local storage.
